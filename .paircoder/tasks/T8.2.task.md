# Implementation Plan

1. Create `ErrorBoundary.tsx`:
   - React class component (error boundaries require class components)
   - Catches render errors, shows friendly message + "Try Again" button
   - Logs error info for debugging
   - Reset on navigation
2. Create `EmptyState.tsx`:
   - Reusable component: icon, title, description, optional CTA button
   - Used for: no jobs found, no plan data, empty barriers
3. Improve existing `PlanSkeleton` in plan page:
   - Match actual layout structure more closely
   - Add pulse animation timing variety
4. Add error/loading states to assess page:
   - Loading spinner during session creation
   - Error message if API fails
5. Wrap app layout with ErrorBoundary
