# Objective

Job match cards showing title, company, transit accessibility, credit check requirements, and application steps. TransitInfo sub-component shows bus route details and Sunday/night warnings.

# Files to Update

| File | Change |
|------|--------|

| frontend/src/components/plan/TransitInfo.tsx | Bus route, schedule, Sunday/night warnings |

# Implementation Plan

1. JobMatch card:
   - Title, company, location
   - Transit badge: green "Bus Accessible" or amber "Requires Transportation"
   - Credit badge: if credit_check_required, show status (eligible now / after repair)
   - Application steps: numbered list from backend enrichment
   - "Apply" link if URL available
2. TransitInfo sub-component:
   - Route number and name
   - Schedule note
   - Warning badges: "No Sunday Service", "No Late Night Service"
   - Coral warning color for restrictions
