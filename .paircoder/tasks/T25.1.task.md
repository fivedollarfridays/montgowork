# Objective

Create the benefits cliff calculator module that computes net income at each wage level for a given household profile. This is WorkPath's key differentiator — showing users that "taking this $15/hr job costs you $400/month in benefits."

Models six Alabama-specific benefit programs with phase-out curves and identifies cliff points where net income drops as wages increase.

# Files to Create

| File | Purpose |
|------|---------|
| `backend/app/modules/benefits/__init__.py` | Module init |
| `backend/app/modules/benefits/types.py` | BenefitsProfile, CliffAnalysis, ProgramBenefit, CliffPoint models |
| `backend/app/modules/benefits/cliff_calculator.py` | Core calculation engine |
| `backend/app/modules/benefits/thresholds.py` | Alabama-specific FPL, SMI, AMI constants and program rules |
| `backend/tests/test_benefits_cliff.py` | Tests for cliff calculator |

# Implementation Plan

## 1. Types (`types.py`, ~60 lines)

- `BenefitsProfile`: household_size (1-8), current_monthly_income, enrolled_programs list, dependents_under_6, dependents_6_to_17, state="AL"
- `ProgramBenefit`: program name, monthly_value, eligible flag, phase_out_start/end (annual income), cliff_type ("gradual" | "hard")
- `CliffPoint`: hourly_wage, annual_income, net_monthly_income, lost_program, monthly_loss, severity ("mild" <$50, "moderate" $50-200, "severe" >$200)
- `CliffAnalysis`: wage_steps list (wage, gross_monthly, benefits_total, net_monthly), cliff_points list, current_net_monthly, programs list, worst_cliff_wage, safe_wage_floor

## 2. Thresholds (`thresholds.py`, ~80 lines)

Alabama-specific 2026 constants:
- FPL by household size (1-8): $15,600 to $54,700
- SNAP: 130% FPL gross / 100% FPL net, max benefit by household size
- TANF: $215/month max (family of 3), 60-month lifetime
- Medicaid: NO expansion in AL — children only via ALL Kids (317% FPL)
- Childcare subsidy: 85% SMI, copay scale
- Section 8: 50% AMI Montgomery MSA, 30% income toward rent
- LIHEAP: 150% FPL, seasonal (~$75/month average)

## 3. Calculator (`cliff_calculator.py`, ~120 lines)

- `calculate_cliff_analysis(profile) -> CliffAnalysis`: Compute net income at $8-$25/hr ($0.50 steps)
- Per-program benefit calculators: `_calculate_snap_benefit`, `_calculate_tanf_benefit`, `_calculate_medicaid_value`, `_calculate_childcare_subsidy`, `_calculate_housing_subsidy`, `_calculate_liheap_benefit`
- Tax estimation: FICA 7.65% + simplified income tax
- Cliff detection: compare adjacent wage steps, flag drops
- `_classify_cliff_severity(monthly_loss) -> str`

## 4. Tests

- Single person, no programs → no cliff
- Family of 3 on SNAP → cliff at ~130% FPL
- Family on SNAP + Childcare → compound cliff
- Each program individually → correct phase-out
- All programs combined → worst-case
- Edge cases: household_size=1, income=0, empty programs
- Cliff severity classification
