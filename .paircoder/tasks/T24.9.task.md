# Objective

Extend the startup warning (currently Anthropic-only, main.py:29) to cover all
configured LLM providers, and make missing API keys auto-fall-back to the mock
provider at request time instead of crashing. Zero API keys = working local dev
out of the box.

# Current State

- `main.py:29` warns only if `ANTHROPIC_API_KEY` is missing — ignores openai/gemini
- `llm_client._validate_key()` raises `ValueError` on missing key → 500 at request time
- `_mock_stream` exists but is only used when `LLM_PROVIDER=mock` is set explicitly

# Target Behaviour

| Scenario | Result |
|----------|--------|
| Configured provider key present | Real provider used (unchanged) |
| Configured provider key missing | WARNING logged at startup + fallback to mock at request time |
| `LLM_PROVIDER=mock` explicit | Mock used, no key needed (unchanged) |
| Unknown `LLM_PROVIDER` value | `ValueError` at startup (explicit misconfiguration) |

# Implementation Plan

1. **`main.py`** — replace line 28-29 with provider-aware check:
   ```python
   _KEY_MAP = {
       "anthropic": (lambda s: s.anthropic_api_key, "ANTHROPIC_API_KEY"),
       "openai":    (lambda s: s.openai_api_key,    "OPENAI_API_KEY"),
       "gemini":    (lambda s: s.gemini_api_key,    "GEMINI_API_KEY"),
   }
   provider = settings.llm_provider
   if provider in _KEY_MAP:
       getter, env_var = _KEY_MAP[provider]
       if not getter(settings):
           logger.warning("%s not set — LLM will fall back to mock responses", env_var)
   ```

2. **`llm_client.py`** — replace `_validate_key` (raises) with `_resolve_provider`
   (returns resolved provider name):
   ```python
   def _resolve_provider(settings, provider: str) -> str:
       if provider == "mock" or provider not in _REAL_PROVIDERS:
           return provider
       key = _KEY_MAP[provider](settings)
       if not key:
           logger.warning(
               "%s is not set — falling back to mock LLM (set LLM_PROVIDER=mock to silence)",
               _ENV_VAR[provider],
           )
           return "mock"
       return provider
   ```

3. **`.env.example`** — add `LLM_PROVIDER=mock` as the zero-config default with comment
