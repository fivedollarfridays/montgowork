# Objective

Implement the assessment route in `routes/assessment.py`. Replace all 3 `NotImplementedError` stubs with real logic. This route creates sessions and kicks off the matching pipeline.

# Implementation Plan

1. `determine_severity(barrier_count)` -> BarrierSeverity:
   - 3+ barriers = HIGH, 2 = MEDIUM, 1 = LOW
2. `extract_primary_barriers(barriers)` -> list[BarrierType]:
   - Return list of BarrierType enums where value is True
3. `create_assessment(request)`:
   - Generate UUID session_id
   - Count barriers, call determine_severity()
   - Call extract_primary_barriers()
   - Set flags: needs_credit_assessment (credit barrier True), transit_dependent (no vehicle + transportation barrier)
   - Build UserProfile
   - Store session in SQLite via query layer (UUID + 24h expiry)
   - Call `generate_plan()` from matching engine
   - Return response with session_id, profile, and plan
4. Add DB dependency injection via `Depends(get_db)`
