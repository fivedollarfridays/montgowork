# Objective

Display estimated commute times on job cards in the plan page. Show drive time, transit time (if routes available), and walk time (if close enough) alongside existing transit info.

# Files to Update

| File | Change |
|------|--------|
| `frontend/src/lib/types.ts` | Add `CommuteEstimate` interface, add `commute_estimate` to `ScoredJobMatch` |
| `frontend/src/components/plan/JobMatchCard.tsx` | Display commute badges in location row |
| `frontend/src/components/plan/TransitInfoDisplay.tsx` | Add commute summary line if available |
| `frontend/src/components/plan/__tests__/JobMatchCard.test.tsx` | Tests for commute display |

# Implementation Plan

1. Add TypeScript types:
   ```typescript
   interface CommuteEstimate {
     drive_min: number | null;
     transit_min: number | null;
     walk_min: number | null;
   }
   ```
2. Add `commute_estimate?: CommuteEstimate` to `ScoredJobMatch`
3. In `JobMatchCard.tsx` location row, add compact badges:
   - Car icon + "~12 min" (drive)
   - Bus icon + "~25 min" (transit, if available)
   - Walking icon + "~8 min" (walk, if <2mi)
4. Use muted-foreground styling, keep compact
5. Accessible: aria-labels on commute badges
