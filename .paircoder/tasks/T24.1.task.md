# Schema

```sql
CREATE TABLE barriers (
    id TEXT PRIMARY KEY,           -- e.g., "CHILDCARE_EVENING"
    name TEXT NOT NULL,
    category TEXT NOT NULL,        -- "childcare"|"transportation"|"credit"|"housing"|"health"|"training"|"criminal"
    description TEXT,
    playbook TEXT,                 -- Best-practice action text (used as RAG corpus document)
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE barrier_relationships (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    source_barrier_id TEXT NOT NULL REFERENCES barriers(id),
    target_barrier_id TEXT NOT NULL REFERENCES barriers(id),
    relationship_type TEXT NOT NULL CHECK(relationship_type IN ('CAUSES','WORSENS','PRE_REQ_FOR','MITIGATED_BY')),
    weight REAL DEFAULT 1.0,
    UNIQUE(source_barrier_id, target_barrier_id, relationship_type)
);

CREATE TABLE barrier_resources (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    barrier_id TEXT NOT NULL REFERENCES barriers(id),
    resource_id INTEGER NOT NULL REFERENCES resources(id),
    impact_strength REAL NOT NULL CHECK(impact_strength BETWEEN 0.0 AND 1.0),
    notes TEXT,
    UNIQUE(barrier_id, resource_id)
);
```

# Seed Data Minimum Coverage

- **Barriers (≥30)**: CHILDCARE_EVENING, CHILDCARE_DAY, TRANSPORTATION_NO_CAR, TRANSPORTATION_NO_TRANSIT, CREDIT_LOW_SCORE, CREDIT_NO_HISTORY, HOUSING_UNSTABLE, HEALTH_NO_INSURANCE, TRAINING_NO_CERT, CRIMINAL_RECORD (plus sub-variants)
- **Edges (≥50)**: Core chain: CHILDCARE → LIMITED_HOURS → LOW_INCOME → CREDIT_ISSUES → HOUSING
- **Playbooks**: ≥1 paragraph per barrier (these become RAG documents in T24.3)
