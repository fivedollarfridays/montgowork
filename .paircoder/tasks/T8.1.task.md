# Objective

Allow users to email their plan to themselves using EmailJS free tier (200 emails/month). No backend email server needed.

# Files to Update

| File | Action |
|------|--------|
| `frontend/src/components/plan/EmailExport.tsx` | NEW: email dialog + send logic |
| `frontend/src/app/plan/page.tsx` | MODIFY: add EmailExport to plan view |
| `frontend/package.json` | MODIFY: add @emailjs/browser dependency |
| `frontend/.env.local` | MODIFY: add NEXT_PUBLIC_EMAILJS_* keys |

# Implementation Plan

1. Install `@emailjs/browser` — `cd frontend && npm install @emailjs/browser`
2. Set up EmailJS account: service, template, public key
3. Create `EmailExport.tsx`:
   - "Email My Plan" button
   - Dialog/modal with email input field (validated)
   - On submit: `emailjs.send()` with plan summary as template params
   - Success/error toast feedback
4. Template params: `{to_email, plan_summary, barriers, job_matches, next_steps}`
5. Wire into `plan/page.tsx` next to the PDF export button
6. Environment variables:
   - `NEXT_PUBLIC_EMAILJS_SERVICE_ID`
   - `NEXT_PUBLIC_EMAILJS_TEMPLATE_ID`
   - `NEXT_PUBLIC_EMAILJS_PUBLIC_KEY`

# Acceptance Criteria
