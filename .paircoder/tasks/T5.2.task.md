# Objective

Build the barrier selection form — a grid of barrier checkboxes with Lucide icons and a live severity preview. First interactive step in the assessment wizard.

# Files to Update

| File | Change |
|------|--------|
| frontend/src/components/wizard/BarrierForm.tsx | Checkbox grid for barrier types with icons + severity indicator |

# Implementation Plan

1. Grid of barrier options matching backend BarrierType enum:
   - Credit (CreditCard icon), Transportation (Bus icon), Childcare (Baby icon)
   - Training (GraduationCap icon), Housing (Home icon), Criminal Record (Shield icon), Health (Heart icon)
2. Each barrier: Card with Checkbox, icon, label, and short description
3. Live severity preview: low (1, green), medium (2, amber), high (3+, coral)
4. Zip code input and employment status Select above barrier grid
5. Work history textarea below barrier grid
6. has_vehicle toggle (affects transit matching)
