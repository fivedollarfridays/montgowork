# Objective

Fix critical backend security and correctness issues: remove hardcoded API key default, enforce session expiry, fix implicit None return, correct HTTP status code, and add startup validation.

# Files to Update

| File | Change |
|------|--------|
| backend/app/core/config.py | Remove hardcoded `credit_api_key = "montgowork-dev"` default |
| backend/app/core/queries.py | Add `expires_at` check to `get_session_by_id()` |
| backend/app/integrations/brightdata/client.py | Add explicit raise after status code checks |
| backend/app/routes/assessment.py | Add `status_code=201` to POST decorator |
| backend/app/main.py | Add startup warning if `anthropic_api_key` is empty |

# Implementation Plan

1. **config.py**: Change `credit_api_key: str = "montgowork-dev"` to `credit_api_key: str = ""`
2. **queries.py**: Add WHERE clause `AND (expires_at IS NULL OR expires_at > :now)` to get_session_by_id
3. **client.py**: Ensure _raise_api_error always raises (add type annotation or assert)
4. **assessment.py**: `@router.post("/", status_code=201)`
5. **main.py**: `if not settings.anthropic_api_key: logger.warning("...")`
