# Objective

Build CreditForm.tsx — a guided credit self-assessment component that collects credit profile data with friendly UX (sliders, dropdowns, helper text) for embedding in the assessment wizard.

# Implementation Plan

- Create src/components/wizard/CreditForm.tsx
- Collect fields matching CreditProfileRequest:
  - current_score: slider or number input (300-850) with score band label (excellent/good/fair/poor/very_poor)
  - overall_utilization: slider (0-100%) with color indicator (green < 30%, amber 30-50%, red > 50%)
  - payment_history_pct: slider (0-100%) with explanation text
  - average_account_age_months: dropdown with friendly ranges (< 6 months, 6-12 months, 1-3 years, 3-7 years, 7+ years)
  - account_summary: simplified fields — total_accounts, open_accounts, collection_accounts
  - negative_items: multi-select checklist (late payments, collections, bankruptcy, charge-offs, liens, judgments)
- Show real-time "Credit Health" indicator based on score + utilization
- Use shadcn/ui Slider, Input, Select components
- Export canAdvance check (score filled + utilization filled + at least accounts filled)
- Match existing wizard step styling (same card layout, colors, spacing as BarrierForm)

# Target Files

- frontend/src/components/wizard/CreditForm.tsx (new)
- frontend/src/lib/types.ts (add CreditFormData interface if needed)
