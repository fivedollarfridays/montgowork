# Objective

Build a reusable multi-step wizard shell with step indicator, progress bar, and prev/next navigation. Used by the assessment flow (assess/page.tsx). Supports 3-4 steps with transitions.

# Files to Update

| File | Change |
|------|--------|
| frontend/src/components/wizard/WizardShell.tsx | Multi-step container with progress bar, step indicators, nav buttons |
| frontend/src/components/wizard/WizardStep.tsx | Individual step wrapper |

# Implementation Plan

1. WizardShell accepts `steps: WizardStepConfig[]` with title, icon, component
2. Progress bar using shadcn Progress component (fills based on current step / total)
3. Step indicator: numbered circles with active/completed/pending states
4. Prev/Next buttons with `canAdvance` validation callback
5. Color scheme: Navy primary, Teal accents, completed steps show checkmark

# Acceptance Criteria

- [x] WizardShell renders step indicator showing current position
- [x] Progress bar fills proportionally as user advances
- [x] Prev/Next navigation works, with optional `canAdvance` validation
- [x] Completed steps show checkmark, current step highlighted in Teal
- [x] Responsive layout — stacks on mobile widths
- [x] `npm run build` passes
