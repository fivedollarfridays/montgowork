# Objective

Implement the retrieval function that powers the Barrier Intelligence Assistant. Given a user
session (barriers, ZIP, schedule), it assembles a rich retrieval context by combining:
1. **Graph context**: root barriers via BFS traversal of `barrier_relationships`
2. **Top-N graph-aware resources**: from `get_top_resources_for_barriers()` (T24.2)
3. **Vector retrieval**: hybrid FAISS similarity search with barrier_tags/ZIP metadata filter

Returns a `RetrievalContext` object serialized into the Claude prompt in T24.5.

# Data Flow

```
Session (barriers, zip, schedule)
    ├─→ find_root_barriers(barrier_ids, db)   # BFS on barrier_relationships
    ├─→ get_top_resources_for_barriers(db, root_barriers, n=5)
    ├─→ RagStore.search(enriched_query, barrier_filter=root_barriers, k=8)
    └─→ RetrievalContext(root_barriers, barrier_chain_summary, top_resources, retrieved_docs, ...)
```
