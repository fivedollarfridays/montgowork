# Objective

Implement eligibility filters in `matching/filters.py`. Replace all 4 `NotImplementedError` stubs with real filtering logic. Filters run BEFORE scoring to reduce the candidate pool.

# Implementation Plan

1. `apply_credit_filter(jobs, credit_severity)` -> (eligible_now, eligible_after_repair):
   - HIGH: flag all jobs requiring background checks
   - MEDIUM: flag only finance/government jobs
   - LOW: all eligible now
2. `apply_transit_filter(jobs, routes, user_zip)` -> list[JobMatch]:
   - Populate `transit_accessible` and `route` fields
   - Flag Sunday/night jobs as "requires personal transportation" (M-Transit constraint)
3. `apply_childcare_filter(resources, user_zip, employer_zips)` -> list[Resource]:
   - Filter childcare providers near both home and work locations
4. `get_certification_renewal(work_history, certifications)` -> list[dict]:
   - Parse work history for cert keywords (CNA, CDL, etc.)
   - Match to training programs from seed data
   - Return renewal pathways with steps and timelines
5. Fill ALL existing test stubs -- remove skip decorators, write real test bodies
