# Implementation Plan

1. Write tests: immediate ready, retries then ready, max retries exceeded, FAILED status
2. Add BrightDataTimeoutError to types.py
3. Implement `poll_until_ready(client, snapshot_id, max_retries, base_delay, max_delay)`
4. Backoff formula: `min(base_delay * 2^attempt, max_delay) + uniform(0, 1)`
5. Mock `asyncio.sleep` in tests to avoid real delays
