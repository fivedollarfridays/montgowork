Fix semantic HTML across all pages: correct heading hierarchy, add skip-to-content link, replace role="button" divs with real buttons, ensure all form controls have proper labels, and add landmark regions.

# Implementation Plan

1. **Skip-to-content link**: Add `<a href="#main-content" className="sr-only focus:not-sr-only ...">Skip to main content</a>` in layout.tsx before Header. Add `id="main-content"` to `<main>` in each page.
2. **BarrierForm**: Replace barrier card divs that use `role="button"` + onClick with actual `<button>` elements. Keep same styling via className.
3. **ErrorBoundary**: Add `role="alert"` to the error message container div.
4. **Heading hierarchy audit**: Verify each page has exactly one h1, sections use h2, subsections use h3. Fix any skipped levels.
5. **Form labels**: Verify all inputs in credit/page.tsx have htmlFor+id pairs (already good in assess/page.tsx).
6. **Landmark regions**: Ensure each page has `<main>`, sections use `<section>` with aria-label where content isn't headed.
