# Objective

Modify PVS scoring to use net income instead of gross wages. A $15/hr job causing a $400/month cliff ranks lower than a $13/hr job preserving all benefits. Job cards display benefits impact badges.

# Files to Update

| File | Change |
|------|--------|

| `backend/app/modules/matching/engine.py` | Pass BenefitsProfile to PVS scorer |
| `backend/app/modules/matching/types.py` | Add CliffImpact to ScoredJobMatch |
| `frontend/src/lib/types.ts` | Add CliffImpact type, update ScoredJobMatch |
| `frontend/src/components/plan/JobCard.tsx` (or equivalent) | Display benefits impact badge |

# Implementation Plan

## 1. New Types (`types.py`)

`CliffImpact(BaseModel)`: gross_wage (str, "$15.00/hr"), benefits_change (float, monthly), net_monthly_change (float), has_cliff (bool), severity (str|None), affected_programs (list[str]).

Add `cliff_impact: Optional[CliffImpact] = None` to `ScoredJobMatch`.

## 2. PVS Scorer (`pvs_scorer.py`)

Replace `_score_earnings()` call with `_score_net_income()`:
- If no BenefitsProfile or no enrolled programs → fall back to existing gross earnings scoring
- Calculate net income at job's hourly wage using cliff calculator
- Calculate net income at current wage
- Score = ratio of job_net / current_net (capped at 1.0)
- Build CliffImpact with change details
- Return (score, cliff_impact) tuple
- `NO_PAY_CEILING` still applies for undisclosed salary

Rename `W_EARNINGS` to `W_NET_INCOME` (same 0.35 weight).

## 3. Engine Integration (`engine.py`)

Extract `benefits_profile` from session data. Pass to `compute_pvs()`. Attach `cliff_impact` to each `ScoredJobMatch`.

## 4. Frontend Badge

Job card display:
- No cliff: "No benefits impact" green badge
- Has cliff: "-$400/month in benefits" red/amber badge
- Net change: "+$200/month vs current" or "-$50/month vs current"
- Affected programs listed: "Loses SNAP, Childcare Subsidy"

## 5. Backward Compatibility

- No benefits_profile → existing earnings scoring (all current tests pass)
- `NO_PAY_CEILING` unchanged for undisclosed salary jobs
