# Objective

Create a `PlanExport` component that renders a print-friendly version of the re-entry plan and uses `html2pdf.js` to generate a downloadable PDF. Maria needs to take her plan to the career center.

# Files to Update

| File | Action |

| `frontend/src/app/plan/page.tsx` | MODIFY: add PlanExport to plan view |
| `frontend/package.json` | MODIFY: add html2pdf.js dependency |

# Implementation Plan

1. Install `html2pdf.js` — `cd frontend && npm install html2pdf.js`
2. Create `PlanExport.tsx`:
   - "Download PDF" button using existing `Button` component
   - Hidden print-friendly div with plan summary, barriers, job matches, next steps
   - On click: render hidden div with `html2pdf()` using A4 page settings
   - Options: margins, page breaks between sections, MontGoWork header/footer
3. Add `@media print` CSS in a `print.css` or Tailwind `print:` utilities:
   - Hide nav, buttons, interactive elements
   - Force white background, black text
   - Page break rules for sections
4. Wire into `plan/page.tsx` — render `<PlanExport plan={plan} profile={profile} />` below the plan content
