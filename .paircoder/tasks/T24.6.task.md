# ExplainSteps Component

Parse Claude's numbered response using shadcn/ui `Accordion`:

```tsx
<Accordion type="multiple">
  {steps.map((step, i) => (
    <AccordionItem key={i} value={`step-${i}`}>
      <AccordionTrigger>Step {i+1}: {step.title}</AccordionTrigger>
      <AccordionContent>
        <p className="text-sm text-muted-foreground">{step.why}</p>
        <EvidenceChips resources={step.evidence} />
        {step.barrierTags.map(tag => <Badge key={tag} variant="outline">{tag}</Badge>)}
      </AccordionContent>
    </AccordionItem>
  ))}
</Accordion>
```
